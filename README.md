# ColomboSmile Dental Hospital

Premium React + Vite website for a Colombo-based dental hospital.

## Stack
- React 18 + Vite
- React Router v6
- Tailwind CSS
- react-icons
- Framer Motion

## Setup
```bash
npm install
npm run dev
```

Visit http://localhost:5173

## Build
```bash
npm run build
npm run preview
```

## Structure
- `src/pages/` — all routed pages
- `src/components/` — reusable UI
- `src/data/` — static content (treatments, team, blog, testimonials, faqs)
